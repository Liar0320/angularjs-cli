## Installation

**BEFORE YOU INSTALL:** please read the [prerequisites](#prerequisites)

### Install
```bash
git clone https://github.com/Liar0320/angularjs-cli.git
# 读取配置文件bower.json进行安装
bower i 
# 安装npm包
npm i
```
## Usage

```bash
npx gulp
```
